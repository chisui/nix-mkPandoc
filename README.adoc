
= mkPandoc 

Provides a way to use https://nixos.org/nix/[Nix] to call https://github.com/jgm/pandoc[pandoc].

== usage

The basic usage is:

include::examples/simple.nix[]

For stuff like LaTex-templates, bibliography and filters take a look at the ./examples/[examples] directory.

An example that uses almost all supported features can be found https://github.com/chisui/dakka/blob/master/thesis/default.nix[here].
